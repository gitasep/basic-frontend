0.3 Conditional (Pengandaian)

Pengandaian atau dikenal dengan istilah 'jika-maka' misalnya jika X melakukan A maka mendapatkan B.
3.1 Directive positif v-if
Secara default bernilai true untuk menampilkan data Test
3.2 Directive negatif v-if
Kebalikan, secara default bernilai false untuk menampilkan data Test
3.3 Memberi nilai pada v-if
Secara default bernilai true untuk menampilkan data Test
3.4 Pasangan v-if dan v-else
v-if menunjukan arti kondisi jika ... maka, misal jika kita makan maka kenyang
v-else menunjukan arti kondisi jika tidak ... maka, misal jika kita tidak makan maka lapar Test
